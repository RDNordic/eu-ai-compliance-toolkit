# AI Act Classification Check (Starter)

1. Define intended purpose.
2. Check prohibited practices (Art. 5).
3. Check Annex I product context (Art. 6(1)).
4. Check Annex III standalone use cases (Art. 6(2)).
5. Assess Art. 6(3) exception only where relevant (it cannot be used if the system performs profiling of natural persons).
6. Check transparency obligations (Art. 50).
7. Determine GPAI obligations if model provider role applies.
8. Record evidence in claim-source register.

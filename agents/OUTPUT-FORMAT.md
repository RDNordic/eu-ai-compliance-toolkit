# Output Format

Use this structure for substantive compliance outputs:

## 1) Conclusion
- Short answer.

## 2) Assumptions
- Jurisdiction:
- Role:
- System purpose:

## 3) Analysis
- Key points with references.

## 4) Sources
- Primary links.

## 5) Confidence
- High / Medium / Low
- Why:

## 6) Open Questions
- Items requiring legal/business confirmation.

## 7) Action Plan
1. Immediate
2. Next
3. Optional hardening

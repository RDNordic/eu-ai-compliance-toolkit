# Task Templates

## Template: AI Act Classification
Input:
- Intended purpose
- Users and affected persons
- Deployment context
- Role (provider/deployer/etc.)

Output:
- Prohibited/high-risk/limited/minimal classification
- Article and annex references
- Required obligations and deadlines
- Uncertainties and next checks

## Template: GDPR Lawful Basis
Input:
- Processing stages (collection/training/inference/monitoring)
- Data categories
- Decision impact on individuals

Output:
- Candidate lawful bases by stage
- Art. 9 condition check
- Art. 22 applicability and safeguards
- DPIA trigger analysis

## Template: DPIA Draft
Input:
- System description
- Data flows
- Risks
- Controls

Output:
- Completed sections
- Residual risks
- Prior consultation trigger check (Art. 36)

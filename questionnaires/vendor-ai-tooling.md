# Questionnaire: Vendor AI Tooling

Use this questionnaire before adopting an external AI tool, model API, chatbot platform, or AI-enabled SaaS product.

## Questions

- Vendor and product name
- Intended internal or external use
- Categories of data that users may input
- Whether personal data, confidential information, or source code will be processed
- Where processing is said to occur
- Whether prompts, files, or outputs are used for provider training
- Available retention and deletion controls
- Available admin, audit, and access controls
- Whether subprocessors are disclosed
- Whether security documentation is available
- Whether a DPA or contractual addendum is available
- Whether the product will influence important business or individual decisions

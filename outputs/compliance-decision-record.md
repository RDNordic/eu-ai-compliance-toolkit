# Output Template: Compliance Decision Record

Use this template when a team needs a reusable governance artifact rather than conversational analysis.

## Fields

- Objective
- System or project name
- Date
- Prepared by
- Facts confirmed
- Assumptions
- Applicable workflow(s)
- Key conclusions
- Sources used
- Confidence level
- Open legal or factual questions
- Decision: go / conditional go / no-go / escalate
- Required actions before next stage
- Owner and review date

## Quality Rule

This record should be concise enough for delivery teams to use, but specific enough that legal, privacy, or governance reviewers can challenge it.

# Prompt Pack: DPIA Draft

Use this prompt when you want an assistant to screen whether a DPIA is likely required and draft the structure for a fuller DPIA.

## Prompt

Use this repository to assess whether a DPIA is likely required for my AI use case. Start with the relevant questionnaire, identify the main data flows and risks to individuals, assess whether the use case creates elevated risk, and tell me what sections a fuller DPIA would need.

Output a source-backed analysis and a short decision record.

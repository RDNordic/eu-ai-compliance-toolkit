# Prompt Pack: AI Act Classification

Use this prompt when you want an assistant to classify a use case under the EU AI Act using this repository.

## Prompt

Use this repository as a compliance runtime. Start with `START-HERE.md` and run the AI Act classification path for my use case. Ask only the questions needed to determine:

- intended purpose
- deployment context
- affected persons
- likely role
- likely classification
- immediate obligations and next actions

Use the repo source material, state assumptions explicitly, and output a source-backed analysis followed by a short compliance decision record.

# Prompt Pack: Vendor Risk Review

Use this prompt when you want an assistant to review an external AI vendor or AI-enabled tool.

## Prompt

Use `START-HERE.md` in this repo and run the vendor-risk-review workflow. Ask me the due-diligence questions first, then assess the vendor’s privacy, retention, training, subprocessors, and deployment-risk posture. Produce a source-backed analysis with a go, conditional-go, or no-go recommendation and list missing answers that block confidence.
